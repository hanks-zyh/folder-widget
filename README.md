# Folder Widget

Folder Widget is a simple launcher widget which allow user create folder to put apps for quick launch.


<a href="https://play.google.com/store/apps/details?id=org.totschnig.myexpenses" target="_blank">
<img src="https://play.google.com/intl/en_us/badges/images/generic/en-play-badge.png" alt="Get it on Google Play" height="70"/></a>


### Features
⭐ custom widget size, backgroud color, radius, margins, paddings.  
⭐ custom folder name, text color, text size, text paddings.  
⭐ quick open system apps, quick search app for shortcuts.  
⭐ adaptive icon shape.  
⭐ support icon pack and mask.  
⭐ folder name shadow option.  
⭐ widget has 2x2 3x3 3+4 1x5 1x4 2x3 3x2 2x4 4x2 size.  
⭐ new ui for configuration.  
⭐ support hide folder name
